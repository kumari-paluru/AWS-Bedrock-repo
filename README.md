# AWS-Bedrock-repo
AWS-Bedrock-projects

1. Install openai
   pip install boto3 openai
2. Create request file first-bedrock-demo.py
3. Generate API_KEY in AWS console and copy it.
4. Export environment variables in powershell from visual studio
   $env:OPENAI_API_KEY= "Value"
   $env:OPENAI_BASE_URL="https://bedrock-mantle.<AWS-Region>.api.aws/v1"
5. Run python code python first-bedrock-demo.py
